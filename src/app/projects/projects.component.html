Tarkov Map Selector

Version 1.0.0

Escape from Tarkov is a multiplayer online video game. This website allows users who play the game to select a map to play. The site randomly selects a map from the map pool and displays it for the user. This site also allows users to favorite maps, which give gives an elevated probability of the favortied map being selected.
